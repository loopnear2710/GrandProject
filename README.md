# GrandProject
## Для работы с crm сервером сначала нужно склонировать репозиторий:
```sh
git clone https://github.com/loopnear2710/GrandProject.git
```
## Затем пропишите в консоль следующие команды:
   ```
npm i
```
   ```
npm start
``` 
## Если не запустилось: 
  ```
	npm run build
```
И
   ```
   npm start
```
